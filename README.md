## ✅ Required Concepts

Your project **must** include all of the following:

- [ ] At least **3 variables** that store meaningful data
- [ ] At least **one `input()`** that asks the user for something
- [ ] At least **one type conversion** (`int()`, `float()`, or `str()`)
- [ ] At least **one `if` / `elif` / `else`** statement that changes what happens based on user input or a variable
- [ ] At least **3 `print()` statements** that display output to the user
- [ ] Code that is **readable** — use clear variable names and leave a comment (`#`) explaining each major section

## 🖥️ Getting Started with GitHub Classroom

This assignment is distributed through **GitHub Classroom**. Follow these steps to get your personal copy and submit your work.

### Step 1 — Accept the Assignment
1. Click the **GitHub Classroom assignment link** provided by your teacher in Canvas.
2. Sign in to GitHub (create a free account at [github.com](https://github.com) if you don't have one).
3. Click **Accept this assignment**. GitHub will automatically create a personal repository for you.
4. Your repo will be named something like `python-final-2026-YourUsername`. This is YOUR copy.

### Step 2 — Edit Your Code

**Option A — Edit directly on GitHub (easiest):**
1. Open your repository on GitHub.
2. Click on `main.py`.
3. Click the **pencil icon** (✏️) in the top right to edit.
4. Write your code, then scroll down and click **Commit changes**.

**Option B — Use a browser-based editor:**
1. On your repository page, press the period key (`.`) on your keyboard.
2. This opens **github.dev**, a full code editor in your browser — no download needed.
3. Edit `main.py`, then click the **Source Control** icon on the left sidebar.
4. Type a commit message and click **Commit & Push**.

### Step 3 — Fill Out Your Planning Doc
1. Click on `PLANNING.md` in your repository.
2. Edit it (pencil icon) and fill in all sections **before you start coding**.
3. Commit your changes.

### Step 4 — Submit
1. Make sure all your changes are committed (saved) to your repository.
2. Copy the URL of your GitHub repository (it should look like `https://github.com/ECCCompSci/python-final-2026-YourUsername`).
3. Paste that link into the **Canvas assignment submission box** and submit.

> ⚠️ **Important:** Your teacher can see every commit you make. Make sure your final code is committed before the deadline!

## 🚀 Tips for Success

- **Plan before you code.** Five minutes of planning saves twenty minutes of debugging.
- **Test as you go.** Run your code after each new section.
- **Use comments.** Add a `#` comment above each section so your code is easy to read.
- **Variable names matter.** `score` is better than `x`. `player_name` is better than `n`.
- **Commit often** — every time you finish a section, save your work with a commit.
- **Ask for help early** — don't wait until the last minute!
